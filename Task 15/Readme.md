# E-commerce Sales Analysis

## Overview

This project generates a random dataset using NumPy and performs basic sales analysis.

## Tools Used

* Python
* NumPy
* Matplotlib
* Jupyter Notebook

## Dataset

* 30 days of sales data
* 10 products
* Random sales values (0–50)
* Random product prices

## Features

* Calculate total revenue
* Find best sales day
* Find best and worst product
* Calculate average revenue
* Plot daily revenue graph

## How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Open Jupyter Notebook:
   jupyter notebook

3. Run the notebook file
